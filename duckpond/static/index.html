What is DuckPond?

A self-hosted analytics platform that gives each of your customers their own isolated data warehouse. Built on DuckDB for fast columnar queries. Deploy to your cloud, query gigabytes in milliseconds, pay only for your infrastructure - not per query.

Everything You Need, Nothing You Don't

Predictable Costs

Deploy to your own AWS/GCP/Azure. No per-query fees, no surprise bills, no data egress charges. You own your infrastructure and your data never leaves your VPC.

Fast Out of the Box

DuckDB's columnar engine queries millions of rows in under 100ms. No cluster to configure, no Spark complexity. Just fast SQL on a single server that handles 100+ concurrent queries.

True Multi-Tenant Isolation

Each customer gets their own DuckLake catalog with isolated storage, API keys, and quotas. No complex RBAC configuration. No noisy neighbor problems. Built for embedded analytics from day one.

Deploy in Hours, Not Months

Docker containers with resource limits, health monitoring, and automatic cleanup. PostgreSQL or SQLite for metadata. Start with a few tenants, add more as you grow.

Flexible Data Ingestion

CSV/JSON/Parquet file uploads with auto-conversion. Arrow IPC streaming for high-throughput ingestion. Prometheus remote write for metrics. All data lands in your catalog, queryable via SQL immediately.

Interactive Notebooks

Built-in marimo notebooks for data exploration and analysis. Python, SQL, and visualizations in one place. Docker-isolated execution with resource controls. Perfect for data teams without DevOps overhead.
