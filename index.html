Everything You Need, Nothing You Don't

Zero Cloud Costs

Self-hosted on your infrastructure. No per-query fees, no surprise bills. SQLite or PostgreSQL for metadata, local storage for data. You own everything.

Stupid Fast

DuckDB's columnar engine processes Parquet at memory speed. Query millions of rows in milliseconds. Single process, no cluster setup, no Spark drama.

DuckLake Catalogs

Each tenant gets their own DuckLake catalog in SQLite (PostgreSQL support coming). ACID transactions, schema versioning, and proper data management. Not just files in folders.

Multi-Tenant from Day One

Isolated accounts with their own catalogs, storage paths, and API keys. Configurable quotas. Give each customer their own space without complex RBAC configuration.

Upload & Stream

Upload CSV/JSON/Parquet files, auto-convert to Parquet. Stream Arrow IPC or Prometheus remote write for telemetry. Files land in your catalog, ready to query.

Query Everything

All data in your DuckLake catalog is queryable via SQL. DuckDB's fast analytical engine handles joins, aggregations, and window functions. Export as JSON, CSV, or Parquet.

# Upload a CSV file
curl -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@sales_data.csv" \
  -F "dataset=sales" \
  http://localhost:8000/upload

# Query your data with SQL
curl -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "sql": "SELECT region, SUM(revenue) FROM catalog.sales GROUP BY region",
    "output_format": "json"
  }' \
  http://localhost:8000/query
